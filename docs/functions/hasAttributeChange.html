    Function hasAttributeChange

    • hasAttributeChange(
          diffResult: DiffResult,
          attribute: string,
          changeTypes?: AttributeChangeType | AttributeChangeType[],
      ): boolean

      Checks if a specific attribute has a specific type of change. Supports dot notation for nested attributes (e.g., "preferences.theme").

      Matching behavior:

      • Exact match: "preferences.theme" matches changes to "preferences.theme"
      • Parent match: "preferences" matches changes to "preferences" OR any nested path like "preferences.theme"
      • Child match: "preferences.theme" does NOT match changes to just "preferences"

      Parameters

      • diffResult: DiffResult

        The result from diffAttributes

      • attribute: string

        The attribute name or path to check (supports dot notation)

      • OptionalchangeTypes: AttributeChangeType | AttributeChangeType[]

        The change type(s) to match (OR logic if array)

      Returns boolean

      true if the attribute has any of the specified change types
