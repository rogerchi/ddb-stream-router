    defer

    • defer(id: string, options?: DeferOptions): StreamRouter<V>

      Mark this handler as deferred - enqueues to SQS from stream, executes from SQS.

      Parameters

      • id: string

        A unique identifier for this deferred handler. This ID is used to match SQS messages back to the correct handler. Must be unique across all deferred handlers in the router.

      • Optionaloptions: DeferOptions

        Optional defer configuration (queue URL, delay seconds)

        Note: When used with batch handlers, each matching record is enqueued individually to SQS. The SQS Lambda trigger's batch settings will determine how records are grouped when processed. This means records that would have been processed together in a non-deferred batch handler may be split across multiple SQS batches. There is no guarantee that related records will be processed together.

      Returns StreamRouter<V>
